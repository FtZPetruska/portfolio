    Lab Class Teaching Assistant

    Université Laval

    Autumn Semesters 2020-2022

    Present, provide support, and correct the lab classes for the Network for Engineers class (GLO-2000). The lab classes provide a more practical approach to networking, assignments use a variety of technologies to explore the layers of the OSI model:

    • Wireshark is used to capture network traffic, students are taught how to analyse the intercepted packets.
    • Nping and Ncat are used to forge traffic, create simple clients and servers, and explore how hosts respond to combinations of parameters. This allows students to manipulate directly the Data Link, Network and Transport layers.
    • Python is used for socket programming. Students are tasked to implement a client and a server allowing them to explore the Application layer.

    cottage Personal Projects

    trace_extractor

    The UdpTraceClient class of the ns-3 simulator uses "Traces" of MP4 videos. These traces consists in a text file containing information for every frame. Aside from the handful of examples, there was no cross-platform tools to create your own trace file, so I created this python wrapper around ffprobe to automate the process!

    SDL Installation Tutorial

    One very common issue with newcomers trying to setup SDL2 is that they often run into issues. This is why I decided to write my own tutorial to install SDL2 and its satellite libraries on Windows, macOS and Linux!

    Yet Another C++ Project Template

    A template C++ project inspired by Jason Turner's. It aims to only provide a subset of features, with a different approach to option checking in CMake.

    This project uses CMake and several tools to ensure quality and cross-platform compatibility throughout the development. The tools available are: VCPKG to acquire your dependencies, Clang-tidy to run static analysis, Clang-format to ensure formatting is respected at build time, Address and Undefined Behaviour Sanitisers when available, Catch2 unit testing, and all the relevant warnings the compiler provides.
